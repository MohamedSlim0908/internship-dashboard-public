## Tell us about a project where you used Python. What part did you personally build or solve?

One strong example is VoiceBubble, a macOS push-to-talk dictation app I built in Python. I personally implemented the flow that records speech, runs local MLX Whisper transcription, sends the transcript through a Claude-based refinement step, stores app-specific context in SQLite, and pastes the improved text back through macOS Accessibility APIs. The main problem I solved was turning rough speech into usable, context-aware text while keeping the user interaction fast and lightweight.

## Describe a time you worked with a REST API (building or consuming). What did you do and what tools did you use?

On Mafhoum, a full-stack community learning platform I co-founded, I helped design and implement more than 50 REST API endpoints across authentication, communities, courses, gamification, and payments. I used Express.js, Prisma, PostgreSQL, Redis, JWT/OAuth authentication, Swagger/OpenAPI-style API validation, and Dockerized local environments. I worked on endpoint structure, database-backed behavior, authentication flows, and client-server integration so frontend features could reliably consume the API.
